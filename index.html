What kind of program is git?

A version control program

What is the git help command?

git help

What is the command for showing the status of the repository?

git status

Command to undo a git add of a file

git reset [filename]

What is the command for amending the last commit?

git commit --amend

What is the command for setting a name that is identifiable for credit when reviewing version history?

git config --global user.name "[firstname lastname]"

What is the command for setting an email address that will be associated with each history marker?

git config --global user.email "[valid-email]"
